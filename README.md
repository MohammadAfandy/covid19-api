# COVID-19 BACKEND

## About This Repo

This repo is the backend of [covid19-next](https://github.com/MohammadAfandy/covid19-next)

### Run Project

Install Dependencies
```
npm install
```

Copy and Edit .env file
```
cp .env.example .env
vi .env
```

#### Development
```
npm run dev
```

#### Production
```
npm run start
```
